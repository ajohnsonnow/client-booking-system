## üöÄ Quick Start

### 1. Clone and Install

```bash
git clone <your-repo-url>
cd <project-folder>
git checkout generic-version
npm install
```

### 2. Configure Your Business

Copy the configuration example:
```bash
cp config.example.js config.js
```

Edit `config.js` with your business details:
- Business name, location, contact info
- Services and pricing
- Brand colors and styling
- Content sections (about, invitation, etc.)

### 3. Setup Environment

Copy the environment template:
```bash
cp .env.example .env
```

Edit `.env` file:
```env
# Basic Setup
PORT=3000
ADMIN_PASSWORD=your-secure-admin-password

# Encryption (auto-generated if not set)
ENCRYPTION_KEY=<auto-generated-on-first-run>

# Email Configuration (optional - see communications setup)
EMAIL_SERVICE=gmail
EMAIL_USER=your-email@gmail.com
EMAIL_PASS=your-app-specific-password

# SMS Configuration (optional)
TWILIO_ACCOUNT_SID=your-twilio-sid
TWILIO_AUTH_TOKEN=your-twilio-token
TWILIO_PHONE_NUMBER=+1234567890
```

### 4. Customize Content

Edit the HTML files in the `public/` directory:

- **index.html** - Main landing page (replace placeholders with your content)
- **portal.html** - Client booking portal (configure services)
- **privacy.html** - Privacy policy (update with your policies)
- **admin.html** - Admin dashboard (pre-configured, but can customize)

### 5. Add Your Branding

Replace placeholder images:
- `public/hero-image.jpg` - Main hero background image
- `public/practitioner-photo.jpg` - Your professional photo
- `public/space-image.jpg` - Your healing space or office

Update colors in `public/styles.css`:
```css
:root {
  --color-burgundy: #722F37; /* Your primary brand color */
  --color-gold: #D4AF37;     /* Your accent color */
  /* ...other color variables... */
}
```

### 6. Run the Application

```bash
npm start
```

Visit http://localhost:3000

## üìã Customization Guide

### Content Sections to Update

1. **Landing Page (index.html)**
   - Hero section: Business name, tagline, location
   - About section: Your story, training, philosophy
   - Invitation section: Your unique message to clients
   - Services: Your offerings, pricing, descriptions
   - FAQs: Common questions specific to your practice
   - Testimonials: Client reviews (or remove section)

2. **Client Portal (portal.html)**
   - Service options and pricing
   - Booking form fields (customize to your needs)
   - Agreements and consent forms

3. **Privacy Policy (privacy.html)**
   - Update with your actual privacy practices
   - Include required legal disclosures for your region
   - Add your contact information

4. **Admin Dashboard (admin.html)**
   - Pre-configured and ready to use
   - Optionally customize branding/colors

### Configuration Options

Edit `config.js` to control:

```javascript
{
  // Core business info
  business: {
    name: "Your Business Name",
    practitioner: "Your Name",
    location: "City, State",
    email: "contact@yourbusiness.com"
  },

  // Visual branding
  branding: {
    logoEmoji: "üå∏",
    primaryColor: "#722F37",
    heroImageUrl: "your-hero.jpg"
  },

  // Your services
  services: [
    {
      name: "Service Name",
      duration: "60 minutes",
      price: "$100",
      description: "Service description"
    }
  ],

  // Feature toggles
  site: {
    passwordProtected: true,      // Require password to view site
    requireInvitationCode: true,  // Require codes to book
    enableTestimonials: true,     // Show testimonials section
    enableFAQs: true,              // Show FAQ section
    enableInquiryForm: true        // Allow new client inquiries
  }
}
```
